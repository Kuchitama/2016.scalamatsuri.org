このセッションでは、Scala plugin for IntelliJ IDEA の内部構造の解説をします。 まず、IntelliJ の Platform API について説明したあと、Scala の型システムについて理解した上で、プラグインで型システムがどのように実装されているかを説明します。 あまり言語の良く知られていない部分や知名度の低いアルゴリズムの話も織り込みます。

Scala に対する深い理解が得られるだけでなく、自分で何かをプラグインとして実装したり、厄介なバグを潰したりできるようになるはずです。 また、カスタム Scala コンパイラプラグインによって独自に拡張した新構文に対応した IntelliJ IDEA のプラグインを実装するのに役立つかもしれません。
