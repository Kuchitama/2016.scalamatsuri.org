Scala が流行っていると聞いたことがある人は Scala の導入を検討しただろう。 Scala の導入を検討することは素晴らしい！でも不幸なことに、たいていの人はくじけることでしょう。

Scala ついて概要を調べると”オブジェクト指向と関数型言語をハイブリッドした静的な型言語”と説明されているのを見かける。 これは技術的に正しいが、しかしあなたの聞きたいことに答えていないだろう。

その代わり、脳内出血を促す驚くようなパラダイム・シフトではないが、 Java より優れていて非常に実用的な Scala の 5 つの強みに焦点をあて、Scala の概要という話題で発表する。 続いて、実用的な Scala の導入における Tips と提言、 そして時間が許せば Scala について自由に質問して頂き回答する時間を設けたい。
