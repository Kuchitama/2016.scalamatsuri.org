Scala には優秀なファイルシステム API がない。お馴染みの scala.io.Source クラスは半分壊れていて、scalax.io プロジェクトは見かけ倒し、java.{io, nio} パッケージはダサいし、scalac や SBT などの中にある、数多くのミニ・ファイルシステム・ライブラリもドキュメント化されておらず中途半端だ。

この現状に甘んじる必要はない。このセッションでは、便利で安定していてドキュメントもあり、あなたのプロジェクトで実際に使えるファイルシステムライブラリである拙作の Ammonite-Ops を紹介する。Ammonite-Ops を使うことによって、これまで冗長で不便だったファイルシステム周りのタスクがどのようにしてシンプルで便利で簡潔になるのか探ることにしよう。
