In this session I will introduce an easy deployment method using sbt-aws, which allows you to manipulate AWS CloudFormation and Elastic Beanstalk from sbt. Using a Play2 application as the sample app, we will create a build that's ready to deploy Docker images pushed to Dockerhub.
