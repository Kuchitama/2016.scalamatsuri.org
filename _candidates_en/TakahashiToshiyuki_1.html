Reading the source code for a framework sounds intimidating, but recent versions of Play has been progressively modularized into compact pieces. Understanding the roles played by each module and the relationships among the modules beforehand will help in Play source code reading tremendously.

In this session, I will introduce the role of each module, and the relationships among the modules as the first step to Play source code reading. This will help you understand the functionality of Play as well its extensibility.
