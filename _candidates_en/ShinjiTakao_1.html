Java VM は、Javaだけでなく様々な言語を実行させるために使われるようになってきています。もちろん Scala もその一つです。

Java VM には、「その中でどのようにメモリが使われているか」など、普段は意識する必要は少ないですが、「もしも」の時のために知っておきたいことがあります。オープンソースのJava VM監視・解析ツール「HeapStats」で「もしも」のときの備えを考えてみましょう。
