このセッションでは、Scala plugin for IntelliJ IDEA の内部構造の解説をします。 まず IntelliJ Platform API から始めて、Scala の型システムの説明をした後で、プラグインの実装の話をします。 あまり言語の良く知られていない部分や知名度の低いアルゴリズムの話も織り込みます。

Scala に対するより深い理解の他に、プラグインを実装する方法を発見できるはずです。これで、自分でバグを潰せるようになります。 また、カスタム Scala コンパイラプラグインによって独自に拡張した新構文に対応した IntelliJ IDEA を実装するのに役立つかもしれません。
