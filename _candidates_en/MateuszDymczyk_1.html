Big data and machine learning are currently all the rage, everyone and their mother wants to do it!

But where do you start? What is ""machine learning"" anyway, how can you use it in your product and how can you benefit from it? Furthermore, traditional techniques are not always suitable for distributed environments so how do you handle big data with them?

This talk will lay out the basics necessary to get you started with the aforementioned topics and present practical, real-life examples and best practices in Scala using technologies such as Spark and its machine learning library MLlib.
