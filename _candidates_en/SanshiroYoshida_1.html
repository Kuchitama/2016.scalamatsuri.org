Although few may be familiar with Prolog, which is a typical logic programming language, many are familiar with SQL that is a query language. Logic programming expresses more general relationships than the query languages in a declarative manner.

Using a logic programming library implemented in Scala and an application using it as examples, I will introduce the Logic monad, which is the essence of logic programming.
