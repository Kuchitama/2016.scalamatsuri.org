DBマイグレーションは、開発者が慎重にオブジェクトモデルとDBスキーマを同期する作業を必要とします。特に、開発中で git ブランチを変えながらの作業は気が狂いそうになります。既存の Scala や Slick の機能を活用して楽で、信頼できて、安全なマイグレーション機能を提供することは可能でしょうか?答えはイエスです。

このセッションでは Scala Forklift のデモを行います。Scala Forklift は、業務に耐えうる、型安全な DBマイグレーションツールで、Slick 開発陣と連携して開発されています。また、実験的機能である Git 連携を使うことでブランチでの作業も楽になります。
